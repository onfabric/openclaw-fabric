## Explore recent interactions
- use the OpenClaw Fabric plugin (id: openclaw-fabric) to explore the users' interactions from today (might be worth checking the days before also to see whether it's a new thing or a daily routine)
- if you find a clear sign of interest try to be useful by looking for relevant information, advice, or even by simply asking a question to clarify what's the user intent
- log your explorations, decisions and user's feedback in your memory so that you do not repeat the same information and learn from the user
